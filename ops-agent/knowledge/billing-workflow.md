# Billing Workflow

Estimate flow: triage the request first, then price it with the tax engine, then draft a brief client message for approval before anything is sent.
Monthly invoicing run: generate invoices for completed jobs, group them by property manager, and hold all sends for human approval.
An estimate is a draft until it is approved; an invoice is not billable until it is finalized.
Every outbound client message and every finalized invoice requires explicit human approval.
